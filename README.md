# ui-vector-oracle-studio

Vector playground — focused single-page studio that serves `vector.buildwithoracle.com`. Extracted from [ui-studio-oracle-studio](https://github.com/Soul-Brews-Studio/ui-studio-oracle-studio) (formerly `oracle-studio`) per issue #28 so the vector UI can evolve independently of the main dashboard.

## Dev

```bash
bun install
bun run dev
```

## Deploy

```bash
bun run deploy
```
