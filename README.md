# vector-oracle-studio

Vector playground — focused single-page studio that serves `vector.buildwithoracle.com`. Extracted from [oracle-studio](https://github.com/Soul-Brews-Studio/oracle-studio) per [oracle-studio#28](https://github.com/Soul-Brews-Studio/oracle-studio/issues/28) so the vector UI can evolve independently of the main dashboard.

## Dev

```bash
bun install
bun run dev
```

## Deploy

```bash
bun run deploy
```
